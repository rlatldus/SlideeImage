## 핵심 기능
- **`이미지조회` :** javascript, React.js 기반 웹서비스 개발.
- **`REACT 라이브러리` :** Slider 관련 프론트 엔드 라이브러리 사용. React.js 최신 framework 사용.
- **`주 사용언어` :** 주 사용언어 : HTML 5, CSS, Javascript, React.js, scss.
